# 1.3.4-tests
1.3.4 «Функциональное тестирование»

- Допустимые символы значения email-адреса:

1. Название:"Ввод email-адреса кириллицей"
Шаги воспроизведения: 
ввести в поле ввода email "тест@mail.ru"
нажать кнопку "Проверить"
Ожидаемый результат: появится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email»
	
2. Название:"Ввод email-адреса, содержащего допустимые символы"
Шаги воспроизведения: 
ввести в поле ввода e-mail "test&test@mail.ru"
нажать кнопку "Проверить"
Ожидаемый результат: появился alert с текстом «Прекрасный email-адрес»

-Соответствие e-mail-адреса паттерну: 

1. Название: "Отсутствие символа @ в email-адресе"
Шаги воспроизведения: 
ввести в поле ввода email "test.test.mail.com"
нажать кнопку "Проверить"
Ожидаемый результат: появится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email»

2. Название: "Отсутствие части email-адреса до символа @"
Шаги воспроизведения: 
ввести в поле ввода email "@mail.ru"
нажать кнопку "Проверить"
Ожидаемый результат:появится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email»

3. Название: "Лишние знаки после имени домена"
Шаги воспроизведения:
ввести в поле ввода email "test@mail.ru="
нажать кнопку "Проверить"
Ожидаемый результат: появится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email»

4. Название: "Введена пустая строка"
Шаги воспроизведения: 
в поле ввода не вводить ни одного символа
нажать кнопку "Проверить"
Ожидаемый результат: появится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email»

- Условия проверки валидации:

1. Название: "Ввод email-адреса в поле инпута без нажатия кнопки "Проверить""
Шаги воспроизведения: 
ввести в поле ввода email "test@mail.ru"
не нажимать кнопку "Проверить"
Ожидаемый результат: alert с текстом «Прекрасный email-адрес не появится, ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email» не появится

2. Название: "Нажатие кнопки "Проверить" несколько раз после ввода валидного email-адреса"
Шаги воспроизведения: 
ввести в поле ввода валидный email "test@mail.ru"
нажать кнопку "Проверить" два или три раза 
Ожидаемый результат: появится alert с текстом «Прекрасный email-адрес» после первого нажатия на кнопку, при последующих нажатиях происходит валидация, но текст сообщения не меняется.

3. Название: "Нажатие кнопки "Проверить" несколько раз после ввода невалидного email-адреса"
Шаги воспроизведения: ввести в поле ввода невалидный email "test@mail.ru="
нажать кнопку "Проверить" два или три раза 
Ожидаемый результат: появится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email» после первого нажатия на кнопку, при последующих нажатиях происходит валидация, но текст сообщения не меняется.

нажать кнопку "Проверить"

ввести в поле ввода невалидный email "test@mail.ru="
нажать кнопку "Проверить"
